# GymOS 🏋️

A modern gym management platform built for fitness studios to manage members, attendance, memberships, and business insights from one place.

## 🚀 Features

### 👥 Member Management
- Add, update, and delete gym members
- Manage member details and membership status
- Search and filter members easily

### 📅 Attendance Management
- Mark members as present
- Track attendance history
- View total visits
- Maintain attendance records

### 📊 Analytics Dashboard
- Revenue insights
- Member growth tracking
- Business statistics
- Visual analytics for better decisions

### ⏰ Membership Expiry Tracking
- Track upcoming expirations
- Identify expiring members
- Manage renewals efficiently

### 📧 Automated Email Reminders
- Send membership expiry reminders
- Notify members before their plan expires
- Built with Resend email API

### 📤 Export Data
- Export member data as CSV
- Download filtered member lists

### 🔐 Secure Access
- Authentication with Clerk
- Gym owner based data isolation
- Subscription based access control

### 🎨 Modern UI
- Responsive dashboard
- Dark and light theme support
- Clean user experience


## 🛠️ Tech Stack

### Frontend
- Next.js
- React
- Tailwind CSS
- Lucide Icons

### Backend
- Next.js API Routes
- MongoDB
- Mongoose

### Authentication
- Clerk

### Email
- Resend
